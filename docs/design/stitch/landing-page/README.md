# FormAuto Hub Landing Page Design Artifact

## Purpose

This folder stores the Stitch-generated Vietnamese landing page design for future frontend work.

Use these files as UI reference only. They are not production source code and do not approve new API, database, payment, Google OAuth, AI, or automation behavior.

## Files

- `screenshots/landing-vi.png`: visual reference exported from Stitch.
- `exports/landing-vi.html`: Stitch HTML export for inspection and layout reference.
- `screen-map.md`: Stitch project and screen metadata.
- `notes.md`: implementation boundaries and review notes.

## Usage Rules

- Treat the design as a marketing landing page reference.
- Keep the app/dashboard UI aligned with `docs/ai/FRONTEND_STYLE_GUIDE.md`.
- Keep product safety language intact.
- Do not implement Deferred features shown or implied by future design variants.
- Validate actual frontend implementation against repo docs, not only this artifact.
