# Screen Map

## Purpose

Reference the FormAuto Hub logo direction using the current app visual style.

## Main Sections

- Primary recommended logo direction.
- Logo mark and wordmark on light backgrounds.
- Icon-only sizing preview.
- Wordmark and compact sidebar usage.
- Favicon and app icon preview.
- Quick usage rules.
- Alternative concepts for comparison.

## Implementation Notes

- Treat the Stitch SVG as a starting design asset, not final production-optimized SVG.
- Before production use, inspect paths, simplify markup, verify alignment at 16px/24px, and export a clean app-ready SVG.
- Use `#2563eb` as the primary brand blue.
- Keep typography close to Geist or the approved app font stack.
- Do not use Google logos, Google Forms branding, or Google four-color identity.

## Phase Alignment

Design artifact only. It can support future frontend branding updates after implementation scope is approved.

## Deferred Items To Avoid

- Trademark/legal approval claims.
- Production asset replacement without frontend implementation approval.
- Google affiliation or Google Forms branding.
- New product features or integration claims.
